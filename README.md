# The Despondent Times

## Description
Welcome to The Despondent Times. A news analytics app that assesses current world actions and measures our moral collapse into a 'moral compass'.

## Concept
Inspired by the [Doomsday Clock](https://thebulletin.org/doomsday-clock/), a metaphorical clock indicating how close we are to atomic war via how many seconds to midnight we are on this metaphorical clock. The Despondent Times uses an API from The Guardian to analyse current news, and build a

## Moral Foundations Theory
The Moral Compass is built on a theory by Jonathan Haidt and his team in 2008 where morality isn't mapped by 'good --> bad', but in six themes from Care—harm, Fairness—cheating, Loyalty—betrayal, Authority—subversion, Sanctity—degradation, Liberty—oppression.

## Technology
Built in p5.js as a framework, and pulling in a JSON data file for the dictionary of words, and The Guardian's free API. Please note that for the purposes of this upload, the API key has been disabled, and uses a JSON of the 50 headlines at the time of upload.

## Hardware
Please view on a laptop or mobile, through Chrome or Firefox. Please ensure you have a mouse and keyboard plugged in for interaction.

## Exhibition setup
To exhibit the work, please create a desk setup, with either:
- A laptop with a keyboard and mouse plugged in
- A laptop plugged into a monitor with a keyboard and mouse plugged in
- A desktop computer with a keyboard or mouse plugged in.

## How to run
Please open the webpage and when prompted press 'start'. Use the 'analyse' button to assess the news, and click it again to return to normal mode. Use the 'show all' button to see all headlines, and 'single' button to view one.
